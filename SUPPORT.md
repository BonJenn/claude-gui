# Support

Blackcrab is an early open source project maintained in spare time.

## Getting Help

- Use GitHub Issues for reproducible bugs and focused feature requests.
- Include your operating system, app version or commit, Claude Code CLI version,
  and reproduction steps.
- For security reports, follow `SECURITY.md` instead of opening a public issue.

## Not Supported

- Private one-on-one setup support.
- Debugging unrelated Claude Code CLI, MCP server, shell, or repository issues.
- Guarantees around unreleased branches or local modifications.
